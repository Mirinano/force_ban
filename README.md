取扱説明書

【警告】

このBOTはBAN権限を操作する強力なBOTです。絶対に悪用してはいけません。

万が一、BOTが暴走したなど、予期せぬ動作をBOTがしている場合はすぐにBOTをBANしてください。

「使い方」

横断BAN
```
$force_ban
対象のユーザーID
```
横断BANの解除
```
$force_unban
対象のユーザーID
```
実行には賛同者が4人、拒否には2人がどちらも必要です。

#black_listのユーザーの一括BAN(BAN解除)

特定のサーバーで#black_listに載っているユーザーをBAN(BAN解除)します。

BAN
```
$past_ban
対象のサーバーID
```
BANの解除
```
$past_unban
